#  Blinkit Performance Dashboard (Power BI)

##  Project Overview
This project demonstrates an **end-to-end data analytics workflow**, where raw retail sales data is transformed into actionable insights using **Power BI, DAX, and data modeling techniques**.

The objective is not just visualization, but to **analyze business performance, identify patterns, and recommend data-driven strategies**.

---

##  Problem Statement
Retail businesses often struggle with:
- Identifying **key revenue drivers**
- Understanding **customer purchase behavior**
- Optimizing **outlet and product performance**

This dashboard addresses these challenges by enabling **interactive and insight-driven analysis**.

---

##  Analytical Approach

### 1. Data Understanding & Preparation
- Cleaned and transformed raw data using **Power Query**
- Handled missing values and ensured data consistency
- Structured data for **efficient modeling and analysis**

### 2. Data Modeling
- Built relationships between datasets
- Designed a **star-schema-like structure** for scalability
- Optimized model for performance and usability

### 3. KPI Development
Defined key business metrics:
- **Total Sales** – Overall revenue performance  
- **Average Sales** – Sales efficiency  
- **Average Rating** – Customer satisfaction indicator  
- **Number of Items** – Inventory scale  

Used **DAX** to create dynamic and reusable measures.

### 4. Data Visualization
- Designed an **interactive dashboard** with slicers for:
  - Outlet Size  
  - Location Type  
  - Item Category  
- Used:
  - Line charts → Trend analysis  
  - Bar charts → Category comparison  
  - Donut charts → Distribution insights  

---

## Key Insights (What the Data Reveals)

###  Sales Trend Analysis
- Sales peaked around **2018**, followed by a decline and stabilization  
- Indicates potential **market saturation or external competition**

###  Geographic Performance
- **Tier 3 cities outperform Tier 1 & Tier 2**
- Suggests **untapped growth potential in non-metro markets**

###  Consumer Behavior
- **Low-fat products contribute ~65% of revenue**
- Reflects a shift toward **health-conscious purchasing decisions**

###  Product Performance
- Top categories:
  - Fruits & Vegetables  
  - Snack Foods  
  - Household Items  
- Indicates dominance of **essential and high-frequency purchase items**

###  Outlet Optimization
- **Medium-sized outlets generate the highest sales**
- Highlights an **optimal balance between scale and efficiency**

---

##  Business Recommendations (Actionable Insights)

- **Market Expansion:** Focus on Tier 3 regions to maximize growth  
- **Product Strategy:** Increase portfolio of health-focused (low-fat) products  
- **Revenue Optimization:** Bundle high-demand items to increase basket size  
- **Operational Efficiency:** Replicate medium outlet model across locations  
- **Customer Experience:** Improve service quality to elevate ratings
